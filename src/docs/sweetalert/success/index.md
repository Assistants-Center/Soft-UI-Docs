# Success

#### **[Parent: sweetalert](/docs/sweetalert/)**

An [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object) which contains all the sidebar information.

## Example Usage

```js

    success: {
        login: "Successfully logged in."
    }
})
```
