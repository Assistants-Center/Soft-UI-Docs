# Command Name

A [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) which is the command name.

## Example Usage

```js
theme: SoftUI({
    commands: [
        {
            list: [
                {
                    commandName: "help"
                }
            ]
        }
    ]
})
```
