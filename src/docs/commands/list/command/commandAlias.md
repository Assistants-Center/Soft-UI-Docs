# Command Alias
#### **[Parent: command](/docs/commands/category/list/command/)**
A [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) which is the command alias.

## Example Usage
```js
theme: SoftUI({
    commands: [
        {
            list: [
                {
                    commandAlias: 'No aliases'
                }
            ],
        }
    ]
})
```
