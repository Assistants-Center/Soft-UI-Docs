# Max

#### **[Parent: graph](/docs/index/graph/)**

A [Number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) which changes the max of the graph.

## Example Usage

```js
theme: SoftUI({
    index: {
        graph: {
            max: 100
        }
    }
})
```
