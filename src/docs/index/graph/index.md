#

#[ThemeConfig].index.graph

#### **[Parent: index](/docs/index/)**

An [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object) which contains the information for the graph.

## Example Usage

```js
theme: SoftUI({
    index: {
        graph: {
            enabled: true,
            lineGraph: false,
            title: "Memory Usage",
            tag: "Memory (MB)",
            max: 100
        }
    }
})
```
