# Error Handler

#### **[Parent: error](/docs/error/)**

A [Function](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function) which contains all the error information.

## Example Usage

```js
theme: SoftUI({
    error: {
        errorHandler: ({ req, res, error, user }) => { ... }
    },
})
```
