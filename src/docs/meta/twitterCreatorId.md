# Twitter Creator ID

### **[Parent: meta](/docs/meta/)**

A [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) which is used for a meta tag.

## Example Usage

```js
theme: SoftUI({
    meta: {
        twitterCreatorId: "PlainDevelopment"
    }
})
```
