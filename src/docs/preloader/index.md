# Preloader
An [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object) which contains all the information for the preloader.

## Example Usage
```js
theme: SoftUI({
    preloader: {
        image: "/img/soft-ui.webp",
        spinner: false,
        text: "Page is loading",
    },
})
```
