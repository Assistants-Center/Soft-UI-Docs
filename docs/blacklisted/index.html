Blacklisted #

An Object which contains all the information for the blacklisted page.

Example Usage #

js
theme: SoftUI({
    blacklisted: {
        title: "Blacklisted",
        subtitle: "Access denied",
        description: "Unfortunately it seems that you have been blacklisted from the dashboard.",
        button: {
            enabled: false,
            text: "Return",
            link: "https://google.com"
        }
    }
})
