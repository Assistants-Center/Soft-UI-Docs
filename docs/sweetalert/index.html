#[ThemeConfig].sweetalert An Object which contains all the SweetAlert information.

Example Usage #

    sweetalert: {
        errors: {
            requirePremium: "You need to be a premium member to do this.",
        },
        success: {
            login: "Successfully logged in.",
        }
    },
})
